# Prompt_Engineering
Házi feladat: 14. Forráskód kommentálása és magyarázat generálása
Az alábbi repositoryn kód fájlok találhatók.
Több féle kód van, mindegyikből két típus.
Egyik egy szimpla kód, másik pedig ugyanaz a kód, csak kikommentelve egy bizonyos stílus és preferencia szerint.
A kódok elején kommentben látható a programozási nyelv, és a kommentelési stílus, ami szerint promptolva lett az MI.
